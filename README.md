<h1 align="left">Discord Clone (Web app)</h1>

###

<p align="left">This is a real time voice and video chat app inspired by Discord!</p>

###

<h2 align="left">Features</h2>

###

<p align="left">- Anonymous authentication with Firebase<br><br>- Send invite links for others to join your channel<br><br>- Have your audio and video instantly transmitted using WebRTC<br><br>- Responsive UI modeled after Discord using React and TailwindCSS<br><br>- Firebase Firestore to enable the exchange of WebRTC SDP offers</p>
